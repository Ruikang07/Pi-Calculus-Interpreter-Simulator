# Pi-Calculus-Interpreter-Simulator
A python implementation of Pi-Calculus Interpreter and Simulator

| Syntax  | Description  |  
| :------------ |:---------------| 
| x<y.P      | write y to channel x | 
| x>y.P      | read from channel x to y       |   
| (x,y,z)@P | bound x,y,z to P        |
